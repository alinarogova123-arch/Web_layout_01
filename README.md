# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Запуск

Для запуска сайта вам понадобится Python третьей версии.

Скачайте код с GitHub. Установите зависимости:

```sh
pip install -r requirements.txt
```

Для запуска сайта необходим файл с каталогом напитков в формате `.xlsx` (файл таблицы excel). Таблица должна иметь следующие столбцы:

| Категория  | Название  | Сорт  | Цена  | Картинка | Акция |
|:-:|---|---|---|---|---|
| Категория напитка | Название напитка  | Сорт Винограда  | Цена продажи  | Ссылка на изображение напитка |Текст для баннера, если проводится акция|

Файл с каталогом можно разместить в директории со скачанным кодом, где находится `main.py`.

В таком случае в качестве пути к файлу можно указать его название и запустить сайт командой:

```sh
python main.py wine.xlsx
```

Если файл с каталогом напитков хранится в другой директории, необходимо указать путь к нему и его название после команды запуска, как в примере ниже:

```sh
python main.py C:\user\documents\wine.xlsx
```

Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).
